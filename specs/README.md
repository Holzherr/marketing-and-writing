# Specifications

This folder contains detailed specifications for all content pieces. Each spec should include:
- Clear objectives
- Target audience
- Key points/messages
- Structure/outline
- Any other relevant context

Use the templates in `/templates/` to create new specs.

## Organization

- `articles/` - Specifications for blog posts and articles
- `social/` - Specifications for social media posts
- `campaigns/` - Specifications for marketing campaigns

## Naming Convention

Use descriptive, kebab-case names:
- Articles: `article-topic-title.md`
- Social: `platform-topic-date.md` or `social-topic-title.md`
- Campaigns: `campaign-name.md`
