# Templates

Templates for creating specs and capturing ideas.

## Available Templates

- `article-spec.md` - Template for article specifications
- `social-post-spec.md` - Template for social media post specifications
- `campaign-spec.md` - Template for campaign specifications
- `idea-template.md` - Template for capturing content ideas

## Usage

1. Copy the relevant template
2. Place it in the appropriate folder (`/specs/` or `/ideas/`)
3. Fill it out with your content details
4. Use it as a reference when creating content
