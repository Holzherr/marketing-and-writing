## Adding New Articles

When adding a new article:
1. Create a markdown file with kebab-case naming: `article-title.md`
2. Include metadata at the top (publication date, outlet, status)
3. Add the article to this index
4. Link to related specs in `/specs/articles/` if applicable

## Naming Convention

Use descriptive, kebab-case names:
- `article-topic-title.md`
- Example: `cooking-to-contracts-lessons-from-founding-two-startups.md`
