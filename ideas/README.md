# Ideas

This folder is for brainstorming and capturing content ideas before they become specs.

## Organization

- `articles/` - Article ideas
- `social/` - Social media post ideas
- `campaigns/` - Campaign ideas

## Workflow

1. Capture ideas here using the template in `/templates/idea-template.md`
2. When ready to develop, create a spec in `/specs/`
3. Link the spec back to the idea

## Naming Convention

Use descriptive names: `idea-topic-title.md`
